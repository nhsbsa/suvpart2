SUV (Single User View) Used in contact centre by agents.

SUV Re-design

Version 1

As per the business process, a new re-design of SUV is required to allow for an improved accessibe version under the new NHS Toolkit.

Presently, the SUV system has various funtions that all appear in a pop-up window which is not considered accessible.

This prototype explores the function of a pop-up function removed and replaced by a seperate page function.

Page List

There is an individual page list, which contains each individual SUV page.

Prototype Journeys

There is a Journey page-list, which contains the Version 1 SUV journey.

Version 2

As per the business process, a new re-design of SUV is required to allow for an improved accessibe version under the new NHS Toolkit.

Presently, the SUV system has various funtions that all appear in a pop-up window which is not considered accessible.

This prototype explores the function of a pop-up function removed and replaced by a seperate page function whilst allowing the user the ability to still see any PCN infomation.

Page List

There is an individual page list, which contains each individual SUV page.

Prototype Journeys

There is a CSS User New Page Function and PCN View PECS, which contains the Version 1 SUV journey.

There is a CSS User New Page Function and PCN View DECS, which contains the Version 1 SUV journey.

There is a Back End View with PCN PECS Info, which contains all patient info and links to all CCS user functions

There is a CSS View with Extend, unlock and close, which contains all patient info and links to all CCS user functions
