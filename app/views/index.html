SUV Re-design

Version 1

As per the business process, a new re-design of SUV is required to allow for an improved accessibe version under the new NHS Toolkit.

Presently, the SUV system has various funtions that all appear in a pop-up window which is not considered accessible.

This prototype explores the function of a pop-up function removed and replaced by a seperate page function.

There is an individual page list, which contains each individual SUV page.

Prototype Journeys

There is a journey page list, which contains the Version 1 SUV journey.

Version 3

In this version of the prototype we have added in a Direct Debit journey. We are to gather more info regarding creating a refund note and flagging a customer as vulnerable.

After a demo run through with the business, a baseline prototype was drafted thus ready for testing.

An in-person testing session was organised with call centre staff on the 19/09/2023 to run through this version of the prototype to see if the design aligned with user mental models.
